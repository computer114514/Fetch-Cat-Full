## 5,修改猫

1,基本信息

请求路径：`/cat/update`

请求方式：`put`

2,请求参数

请求参数样例：/cat/del

~~~json
{
    "catId":"sld1",
    "name":"sleep猫",
    "favorability":3,
    "petCount":1,
    "walkCount":1
}
~~~

3,响应参数

参数格式：`application/json`

~~~json
{
  "code": 200,
  "msg": "success",
  "data": []
}
~~~

不需要返回任何东西，因为如果改变了，乐观也是更新了。
